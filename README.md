## 使用フレームワーク・言語

- JavaScript
- TypeScript
- React.js

## 使用ツール・ライブラリ

- create-react-app
- redux
- react-router
- storybook
- jest
- styled-components
- react-spring

## デプロイ手順

- ビルド `yarn build`
- gh-pages へデプロイ `yarn deploy`
